🔧 トラブルシューティングガイド

問題のパターン:
1. 白いページが表示される → JavaScript エラー、React 読み込み失敗
2. 404 エラー → SPA ルーティング問題
3. データが表示されない → Sanity 接続問題
4. 無限ローディング → API 応答問題

📋 ブラウザ コンソール確認

F12 キーを押して開発者ツールを開き、以下を確認してください:
• Console タブ → JavaScript エラーの確認
• Network タブ → 失敗したリクエストの確認
• Elements タブ → HTML の読み込み状況の確認

🧪 手動テストコマンド

ターミナルで以下を実行してサーバー状態を確認: # サーバー確認 curl http://localhost:3000/ curl http://localhost:3000/blog curl http://localhost:3001/api/blog-posts # プロセス確認 ps aux | grep -E "(node|spa-server)" lsof -i :3000 -i :3001 -i :3333
